# Linear Regression Intuition

## What is Linear Regression?

Linear Regression is a Machine Learning algorithm used to predict continuous values.

Examples:
- House price prediction
- Salary prediction
- Marks prediction

It learns the relationship between input and output data.

---

# Main Equation

```python
y = mx + b
```

Where:
- y = predicted output
- x = input feature
- m = slope
- b = intercept

---

# Important Concepts

## 1. Slope

Slope tells how much the output changes when the input increases.

Example:
If marks increase by 20 for every extra study hour,
then slope = 20.

---

## 2. Intercept

Intercept is the output value when input is zero.

Example:
If a student studies 0 hours and still gets 5 marks,
then intercept = 5.

---

## 3. Best Fit Line

Best fit line is the line that passes closest to all data points.

It shows the relationship between input and output.

---

# Example Dataset

| Hours Studied | Marks |
|---|---|
| 1 | 20 |
| 2 | 40 |
| 3 | 60 |

The model learns that:
More study hours → More marks.
