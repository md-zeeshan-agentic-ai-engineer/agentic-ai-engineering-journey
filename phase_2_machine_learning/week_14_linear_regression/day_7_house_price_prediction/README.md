# House Price Prediction

## Project

This project predicts California house prices using Linear Regression.

## Dataset

California Housing Dataset (Scikit-Learn)

## Libraries

- NumPy
- Pandas
- Scikit-Learn
- Matplotlib

## Workflow

1. Load dataset
2. Explore data
3. Train/Test Split
4. Train Linear Regression model
5. Make predictions
6. Evaluate model
7. Interpret coefficients

## Results

- MAE ≈ 0.533
- MSE ≈ 0.556
- R² Score ≈ 0.576
