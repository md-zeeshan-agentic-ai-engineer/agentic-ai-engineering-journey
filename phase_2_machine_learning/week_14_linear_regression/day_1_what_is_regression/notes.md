# Regression

## Definition of Linear Regression

- Linear Regression is a ML algorithm used to predict continuous numerical values.

## Real-life examples:

- House price prediction
- Salary prediction
- Temperature prediction
- Stock price estimation
- Marks prediction

## Important Terms

### Feature (X)
- Input variable used for prediction.

### Target (y)
- Output variable to predict.

### Dataset
- Collection of training data.

### Training
- Process where model learns patterns from data.

### Prediction
- Model predicts unseen values.

## Regression vs Classification

| Regression | Classification |
|---|---|
| Predicts numbers | Predicts categories |
| House price | Spam detection |
| Salary | Disease Yes/No |

## ML workflow Recap

### Step 1 - Collect Data

Example:
- House prices

### Step 2 - Train Model

- Algorithm learns patterns.

### Step 3 - Predict
