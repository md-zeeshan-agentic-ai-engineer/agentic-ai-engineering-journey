## What is AI?
AI = Systems that simulate human intelligence (learning, reasoning, decision-making)
    Example: Recommendation systems, self-driving cars

## What is ML?
ML = A subset of AI where systems learn patterns from data to make predictions or decisions without explicit programming.
    Example: House price prediction, Spam detection, Customer segmentation, Recommendation systems, AI driven cars, AlphaGo, Robotics


## What is DL?
DL is a subset of ML that uses neural networks with many layers to learn complex patterns.
    Example: Chatbot, Vision related technology



## AI vs ML vs DL

| Feature         | AI                  | ML               | DL                   |
|-----------------|---------------------|------------------|----------------------|
| Definition      | Mimics intelligence | Learns from data | Uses neural networks |
| Data dependency | Low                 | Medium           | High                 |
| Complexity      | Low                 | Medium           | High                 |
| Example         | Chatbot             | Spam filter      | ChatGPT              |

AI
 └── ML
      └── DL

## Types of Machine Learning

### Supervised Learning

- Input + correct output already given.
    Example: House price prediction, Spam detection

- Main goal: Learn Mapping
    x -> y


### Unsupervised Learning

- No lables.
- Goal: Find hidden patterns

    Example: Customer segmentation, Recommendation systems



### Reinforcement Learning

- Agent learns using rewards/punishments.
    Example: Self-driving, AlphaGo, Robotics


## Real-world ML Examples

Netflix recommendation
YouTube suggestions
Google Maps traffic prediction
Fraud detection systems


## Types of problems in ML

Regression -> Predict continuous values
    Example: House price

Classification -> Predict categories
    Example: Spam or Not Spam

## ML Workflow

Data Collection -> Collect raw data from sources
Data Cleaning -> Remove missing/incorrect data
Feature Engineering -> Convert data into useful format
Model Selection -> Choose algorithm
Training -> Train model on data
Evaluation -> Check accuracy/performance
Deployment -> Use model in real-world


Key Insight:
Machine Learning is not about algorithms, it's about understanding data and patterns.


Real Insight:
This same ML pipeline is used in:
- LLM systems
- AI Agents
- Recommendation systems
- Autonomous systems


## My Understanding
Machine Learning is about finding patterns in data and using them to make smart decisions automatically.
