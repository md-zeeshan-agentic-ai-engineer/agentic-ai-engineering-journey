## Goal

## Concept
A function is a mapping from input to output.

f(x) = x²  
Input: x = 2  
Output: 4  

A function takes input, processes it, and returns output.

## Python Examples

### Square Function
```python
def square(x):
    return x**2
```

### Cube Function
```python
def cube(x):
    return x**3
```

### Double Function
```python
def double(x):
    return 2*x
```

## Test
```python
print(square(5))   # 25
print(cube(5))     # 125
print(double(5))   # 10
```

## Deep Understanding
- Function = mapping
- Input → Output transformation

In Machine Learning:
- Model = function
- Input = data
- Output = prediction

Example:
Image → Model → Cat/Dog
