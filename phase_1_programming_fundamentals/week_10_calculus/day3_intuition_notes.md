# Day 3 - Derivative Intuition

## Core Idea
Derivative = slope of curve  
Slope = direction of change

---

## Function 1: f(x) = x^2

Derivative: f'(x) = 2x

Points:
- x = -2 → slope = -4 → decreasing
- x = 0 → slope = 0 → minimum
- x = 2 → slope = 4 → increasing

Behavior:
- Decrease → Minimum → Increase

---

## Function 2: f(x) = x^3

Derivative: f'(x) = 3x^2

Points:
- x = -2 → slope = 12 → increasing
- x = 0 → slope = 0 → flat
- x = 2 → slope = 12 → increasing

Behavior:
- Always increasing
- No minimum or maximum
- Important: slope = 0 does NOT always mean minimum

---

## Function 3: f(x) = sin(x)

Derivative: f'(x) = cos(x)

Points:
- x = 0 → slope = 1 → increasing
- x = π/2 → slope = 0 → maximum
- x = π → slope = -1 → decreasing

Behavior:
- Increase → Maximum → Decrease → Repeat

---

## Key Learnings

- Slope > 0 → increasing
- Slope < 0 → decreasing
- Slope = 0 → turning point (not always min/max)

---

## ML Connection

- Derivative = direction signal
- Used to find minimum of loss function
- Base of Gradient Descent

---

## Final Understanding

Understanding slope = understanding optimization  
This is core of Machine Learning
