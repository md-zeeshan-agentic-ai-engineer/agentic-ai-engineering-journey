# Week 10 — Day 2: Derivatives (CORE)

## 1. What is a Derivative?
A derivative represents the rate of change of a function with respect to a variable.

- It tells how fast the output is changing when input changes.
- Example:
  - Derivative of distance = velocity
  - Derivative of velocity = acceleration

---

## 2. Core Example

Function:
f(x) = x^2

Derivative:
f'(x) = 2x

Interpretation:
- At x = 2 → slope = 4
- At x = 5 → slope = 10

This means the graph becomes steeper as x increases.

---

## 3. Intuition

- Derivative = slope of a curve at a point
- Flat region → derivative ≈ 0
- Increasing function → positive derivative
- Decreasing function → negative derivative

---

## 4. Power Rule

If:
f(x) = x^n

Then:
f'(x) = n * x^(n-1)

Examples:
- x^3 → 3x^2
- x^5 → 5x^4
- 3x^2 → 6x

---

## 5. Machine Learning Connection

Derivatives are used to understand:

- Is loss increasing or decreasing?
- In which direction should the model move?

This leads to optimization techniques like Gradient Descent.

---

## 6. Key Takeaway

Derivative = slope = direction = learning signal in ML

Understanding derivatives is essential for optimization in machine learning.
