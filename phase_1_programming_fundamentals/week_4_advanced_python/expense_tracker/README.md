# Expense Tracker (Python CLI Project)

A simple command line expense tracker built using Python.

This project allows users to:

• Add expenses
• View all expenses
• Calculate total expenses
• Search expenses by category
• Store data in JSON file

---

## Project Structure

expense_tracker/
│
├── main.py
├── utils.py
├── storage.py
└── data.json

---

## How to Run

1 Open terminal

2 Navigate to project folder

3 Run the program

python main.py

---

## Features

1 Add Expense
2 Show Expenses
3 Total Expense
4 Search by Category
5 Exit
