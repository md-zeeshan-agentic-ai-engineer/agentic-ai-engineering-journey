---

## Key Concepts Learned

### Vectors

* A vector represents **magnitude + direction**
* Representation: `[x, y]`
* Used to represent data in ML

---

### Vector Operations

* Addition of vectors
* Scalar multiplication
* Basic manipulation of data

---

### Dot Product (Very Important)

* Formula:
  A · B = |A||B|cosθ
* Measures similarity between vectors
* Used in:

  * Recommendation systems
  * NLP (text similarity)
  * Search engines

---

### Matrices

* 2D representation of numbers
* Collection of vectors
* Core structure in ML models

---

### Matrix Operations

* Addition
* Multiplication
* Matrix multiplication = backbone of neural networks

---

## AI Connection (Most Important)

* Data → represented as **vectors**
* Weights → represented as **matrices**
* Neural Networks → multiple **matrix multiplications**

This means:
**AI = Linear Algebra in action**
